## Getting started

Install dependencies with [npm](https://www.npmjs.com/).

```shell
npm install
```

### Frontend

The frontend has been initialized with [create-react-app](https://create-react-app.dev/).

Starting development server.

```shell
npm run start
```

### Backend

```shell
node backend/server.js
```
